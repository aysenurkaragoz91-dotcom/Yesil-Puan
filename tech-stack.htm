## 🛠️ Önerilen Teknoloji Yığını: "The Speed Stack"

 

| Teknoloji | Görevi | Neden Seçtik? |

| :--- | :--- | :--- |

| **Next.js** | Framework (İskelet) | Hem ön yüzü (HTML) hem arka yüzü (API) tek klasörde halleder. Kurulumu saniyeler sürer. |

| **Tailwind CSS** | Tasarım (Stil) | CSS dosyalarıyla boğuşmadan, HTML içine kelimeler yazarak profesyonel tasarım yapmanı sağlar. |

| **Gemini API** | Yapay Zeka (Beyin) | Markaları analiz eder, puan verir ve yerel alternatifleri akıllıca eşleştirir. |

| **Vercel** | Yayınlama (Hosting) | Kodunu tek tıkla tüm dünyanın erişebileceği bir web sitesine dönüştürür (Ücretsiz). |

 

### 💡 Neden Bu Seçimler?

Başlangıç seviyesinde ayrı bir "Server" (Arka yüz) kurmakla vakit kaybetmemelisin. Next.js, Gemini API anahtarını güvenli bir şekilde saklayabileceğin gizli bir oda (API Routes) sunar. Böylece anahtarın başkalarının eline geçmez.

## 🚀 Kurulum Adımları (Sıfırdan Başla)

 

Bir kıdemli developer gibi terminalini aç ve şu adımları takip et:

 

### 1. Projeyi Oluştur

Terminaline (veya VS Code terminaline) şu komutu yaz:

```bash

npx create-next-app@latest sustaina-track

```

*Sana sorulan sorularda `TypeScript: No`, `Tailwind CSS: Yes`, `App Router: Yes` seçeneklerini seç.*

 

### 2. Proje Klasörüne Gir

```bash

cd sustaina-track

```

 

### 3. Gemini SDK'sını Yükle

Google'ın yapay zekasını projenle konuşturmak için şu paketi ekle:

```bash

npm install @google/generative-ai

```

 

### 4. API Anahtarını Güvenceye Al

Projenin ana dizininde `.env.local` isimli bir dosya oluştur ve içine Google AI Studio'dan aldığın anahtarı yapıştır:

```text

GEMINI_API_KEY=senin_api_anahtarin_buraya

```

## 🧠 Gemini API'yi Uygulamana Bağlama (İlk Kodun)

 

`app/api/chat/route.js` dosyası oluştur ve içine bu mantığı koy. Bu kod, kullanıcının yazdığı markayı Gemini'ye gönderir ve ondan puan + yerel alternatif ister:

 

```javascript

import { GoogleGenerativeAI } from "@google/generative-ai";

 

export async function POST(req) {

  const { brand } = await req.json();

  const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY);

  const model = genAI.getGenerativeModel({ model: "gemini-3-flash" });

 

  const prompt = `Kullanıcı şu markayı arattı: ${brand}.

  Bu markaya 100 üzerinden sürdürülebilirlik puanı ver.

  Eğer puanı 50'den düşükse, Türkiye'den yerel ve daha çevreci bir alternatif marka öner.

  Yanıtı şu formatta ver: Puan: [Puan], Neden: [Kısa açıklama], Alternatif: [Marka Adı].`;

 

  const result = await model.generateContent(prompt);

  return Response.json({ text: result.response.text() });

}

```
